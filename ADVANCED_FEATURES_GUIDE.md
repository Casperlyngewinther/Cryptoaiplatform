### API Endpoints
```bash

{
  "name": "RSI Strategy Backtest",
  "strategy": {
    "name": "RSI Mean Reversion",
    "rules": [
      {
        "type": "CROSS_BELOW",
        "indicator1": "RSI",
        "indicator2": 30,
        "action": "BUY"
      }
    ],
    "indicators": [
      {"type": "RSI", "parameters": {"period": 14}}

  "symbol": "BTC/USDT",
  "timeframe": "1h",
  "startDate": "2024-01-01",
  "endDate": "2024-09-01",
  "initialCapital": 10000

# Get backtest results
GET /api/v2/backtest/results/{backtestId}
```

### Configuration
